Ferramentas utilizadas

As ferramentas utilizadas nesse curso são: Um navegador, browser, de sua preferência e a IDE - VSCode

Inspetor de elementos

Inspetor de elementos é a ferramenta do browser que ajuda a investigar o código html e css com base nos elementos da tela.

Estrutura básica do HTML

A estrutura básica do HTML é composta de 3 tags principais: html, head e body.

A tag html serve para as configurações básica do HTML. A tag head serve para as configurações iniciais da página.

A tag body serve para mostrar aos usuários as informações da página.

Tags básicas de formatação do texto

As tags básicas de formatação em um texto são: Strong, i, temos também a tag de image e a title.

A tag strong serve para colocar em destaque a palavra. A tag i tem a função de colocar em itálico a palavra.

A tag title coloca titulo na aba. A tag de image te permite colocar uma imagem com base na fonte fornecida.

Atributos das tags

Alguns atributos são fundamentais para as tags. Um atributo muito importante para tag é o src , que te permite colocar a fonte da imagem.

Outra tag importante é a id e type. O id permite com que o elemento html seja identificado no css. O type faz com que um campo html possa ser number, text, etc..

Listas ordenadas e não ordenadas

As listas podem ser ordenadas e não ordenadas. As listas ordenadas tem o aspecto de numeração, um padrão definido para um assunto, através da tag ol.

Já as listas não ordenadas, tem o símbolo com uma bolinha e é feita através da tag ul.

Links das páginas

O html te permite adicionar links em suas páginas. A tag "a" te possibilita inserir o link e com o atributo href te levar a algum lugar na página ou a outra página html.

Já o atributo target te mostra qual será a função do teu link, se abre em uma nova guia, se tem outro comportamento, etc.. A mais utilizada é a blank que te permite abrir em outra aba o link.
